SFX-Calc User Guide

Introduction

SFX-Calc is a calculator app designed for academic, scientific and engineering purpose. The calculator features:

  1. Basic arithmetic calculation: Plus, Minus, Multiply, Divide
  2. Calculation with one operand fixed as constant
  3. Calculation with an non-volatile memory storage
  4. Calculation with 10 volatile memory storages
  5. Fraction and percentage calculation
  6. Binary / Octal / Decimal / Hexadecimal calculation
  7. Various functions like Trigonometric, Hyperbolic, Logarithm, Exponential, Power, Root, ... etc.
